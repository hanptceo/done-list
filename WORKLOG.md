## 2026-07-13

### 폰트
- 앱 전역 폰트를 Gmarket Sans로 통일
- IBM Plex Mono 제거, `font-mono`도 Gmarket Sans 사용
- weight 400/600을 Medium/Bold에 매핑

### 한달 화면
- 일평균을 총합 옆에 15px로 `(평균 …/일)` 표기
- 분 단위는 `분`으로 표시
- 루틴은 루틴별 합산, 직접입력은 같은 이름끼리 묶어 합산
- 섹션 제목: 작업별 기록시간

### 네비게이션 · PWA
- 상단 헤더에 `DONE!!DONE!!` 표시
- 하단 고정 탭 메뉴(오늘/주간/한달/루틴/설정)
- 웹앱(홈 화면 추가)용 manifest, 아이콘, service worker 추가
- 하단 메뉴 라벨 `홈` → `오늘`

### 시간 표기
- `formatDurationShort`의 `m` → `분`, `h` → `시간` (예: `1시간 10분`)

### 주간 화면
- 토요 섹션: 차분한 푸른 톤
- 일요 섹션: 차분한 붉은(로즈) 톤
- 섹션 간격 50% 축소 (`space-y-3` → `space-y-1.5`)

### 날짜 표기
- 홈·주간 날짜 타이틀을 `yyyy.mm.dd` → `yyyy.m.d`로 변경
  - 오늘: `2026.7.13 (월)`
  - 주간: `2026.7.13 ~ 7.19`

### 고정루틴
- 등록·수정 시 프리셋 색상 + 직접 색 선택 지원 (기록 모달과 동일)

### 설정 · 데이터
- 데이터 내보내기: 저장 위치 선택 대화상자(`showSaveFilePicker`) 사용
- 미지원 브라우저는 기존 다운로드 방식으로 폴백
- 시작시간 사용 옵션 추가 (기본 꺼짐). 끄면 입력·표시 숨김

### 오늘 화면 · 기록 모달
- `+` 버튼을 화면 공백(남은 영역) 가운데에 배치
- 기록하기(70%) / 닫기(30%): 기록하기는 저장 후 폼 초기화(연속 입력), 닫기는 창 닫기
